# 人物関係図

## メインキャラクター

### グラナイト（24歳）
- 役割：主人公 / 「グラナイト杖店」2代目店主
- 性格：ぶっきらぼうで頑固。自分の技術に絶対の自信
- 能力：素材の微細な歪みや魔力の澱みを感知する「神の指先」

### ネルル（24歳）
- 役割：ヒロイン / 王立魔法具工房 第1設計局 主任
- 性格：高飛車でプライドが高い完璧主義者
- グラナイトとは幼馴染

### シナバー長官（50代）
- 役割：敵対者 / 王立魔法具工房長官
- 正体：魔族と密約を結んだ売国奴
- 30年前、八百長で「英雄」になった

---

## サブキャラクター

### マルカイト
- 役割：魔導具鑑定局 局長（30年のベテラン鑑定士）
- 先代アンバーの旧友・同期
- シナバーに脅されながらも証拠を保管

### アクア
- 役割：若い騎士 / バーミリオンの部下
- 事故の目撃者（緑色の発光を見た）
- グラナイトとネルルの仲介役

### バーミリオン
- 役割：騎士 / 事故の被害者
- 杖暴発で腕が動かなくなり騎士引退
- グラナイトを恨んでいる

### アンバー（故人）
- 役割：グラナイトの父 / 先代店主
- マルカイト・シナバーと王立工房の同期
- 名工として知られていた
