# 108枚連続メモ書き挑戦 - 人生にサプライズを起こす108の問い

## ゼロ秒思考メモ書きの基本ルール

1. **A4用紙を横に使用**（1枚1テーマ）
2. **タイトルを1行目に書く**（20～30字）
3. **4～6行、各20～30字で回答**（合計200字程度）
4. **1テーマ1分で書く**（深く考えず、思いついたことを素早く書く）
5. **毎日10～20枚書く**（習慣化が重要）
6. **書いたら見返さない**（すぐ次のテーマへ）
7. **1週間後に見返す**（新しい気づきが得られる）

## 重要なポイント

- **完璧を求めない**：思いついたことをそのまま書く
- **批判しない**：自分を責めず、事実を書く
- **行動につなげる**：書いた内容から具体的な行動を考える
- **繰り返し書く**：同じテーマを何度でも書いて良い

## 挑戦の進め方（赤羽流実践法）

### 基本の進め方

1. **毎日10～20枚書く**（習慣化が最重要）
2. **1問1分で書く**（深く考えず、思いついたことを素早く書く）
3. **書いたら見返さない**（すぐ次のテーマへ進む）
4. **同じテーマを何度でも書いて良い**（繰り返すことで深まる）
5. **1週間後に見返す**（新しい気づきが得られる）

### 効果を高めるコツ

- **「なぜ」と「どうすれば」を必ず書く**：問いの答えだけでなく、理由と行動を明確にする
- **具体的な行動を書く**：抽象的な答えではなく、「今日からできること」を明確にする
- **批判しない**：自分を責めず、事実を書く
- **完璧を求めない**：思いついたことをそのまま書く
- **行動につなげる**：書いた内容から必ず具体的な行動を考える

### 108枚書き終えた後

1. **すべてのメモを見返す**
2. **共通するテーマやパターンを探す**
3. **最も重要な気づきを3つ選ぶ**
4. **その気づきから具体的な行動計画を立てる**
5. **行動計画を実行する**

## 期待される効果

- **思考の整理**：頭の中のモヤモヤが明確になる
- **悩みの解決**：問題の本質が見えてくる
- **行動の明確化**：何をすべきかがわかる
- **自信の向上**：自分を理解することで自信がつく
- **人生の方向性の明確化**：本当に大切なことがわかる
- **内面的な成長**：深い自己理解が得られる

## よくある質問

**Q: 1分で書けない場合はどうすればいいですか？**
A: 1分を目安に、思いついたことを書けばOKです。完璧を求めず、とにかく書くことが大切です。

**Q: 同じテーマを何度も書いてもいいですか？**
A: はい、何度でも書いてください。繰り返すことで、より深い気づきが得られます。

**Q: 書いたメモはどうすればいいですか？**
A: 1週間後に見返してください。新しい気づきが得られます。108枚書き終えたら、すべてを見返して共通テーマを探してください。

**Q: 行動につなげられない場合はどうすればいいですか？**
A: そのテーマについて、もう一度メモを書いてみてください。「どうすれば」の部分をより具体的に書くことで、行動が見えてきます。
