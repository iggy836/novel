## 殺人トリックの全容

### 犯人の実行手順

1. **準備段階（祝賀会開始前）**
   - リゼットは定期検診を口実に神父の個室へ
   - ヴァルダスと二人きりになる（1時間の空白時間）
   
2. **殺害の実行**
   - リゼットは神父の弱みを握っており、脅迫的な会話
   - 神父が抵抗しようとした際に、リゼットは黒鉄から入手した特殊な導電棒（火かき棒）で攻撃
   - 導電棒に電気魔法を流し、神父の体に電撃を与える
   - 電撃により神父は意識朦朧、体に電気熱傷を負う
   - リゼットは神父を全裸にし、床に倒す
   
3. **遅延トリックの設置**
   - **重要**: リゼット自身はアンデッドであり、体温を感じない
   - 神父の部屋の**暖房システムを停止**する（配管の弁を閉める）
   - 神父は重傷だが即死ではなく、まだ生きている状態
   - リゼットは「永劫の帯留」の指輪を神父の指にはめる
      - この指輪はアンデッド属性を持つアイテム
      - 装着者を疑似的にアンデッド化する効果
   - 祝賀会に戻り、アリバイを確立

### トリックの核心メカニズム

#### phase_1: 暖房停止による遅延
- リゼットが暖房を停止したため、神父の部屋は徐々に冷えていく
- 外は猛吹雪（摂氏マイナス10度以下）
- 神父は重傷で動けず、寒さの中で意識を失いかける

#### phase_2: マリアによる善意の暖房起動
- 消灯前の見回りでマリアが神父の部屋が寒いことに気づく
- ノックして許可を得て（ねぼけた声で了承）
- 地下の暖房室で神父室への熱供給を起動
- ログに記録：『23:42 シスター・マリア → 暖炉 / 火の魔法（小規模）』

#### phase_3: 暖房起動から数分後——悲劇の連鎖
1. **温かい風が配管を通って神父の部屋へ到達**
   - 暖房システムには「癒しの風」機能がある
   - シスターたちが祈りを込めた**聖属性の魔法**が暖気と共に送られる
   
2. **神父の苦しみの始まり**
   - 神父は「永劫の帯留」の指輪により疑似アンデッド状態
   - 聖属性の「癒しの風」がアンデッドに接触
   - **聖属性はアンデッドに激痛を与える**
   - 体中の傷口に聖なる風が染み込み、激しい痛みで目覚める
   
3. **神父の絶叫**
   - 「熱（魔法）をくれ！ いや、熱い！ やめろおおお！！」
   - 矛盾した叫び：寒かったのに「熱い」と叫ぶ
   - 実際は聖属性の痛みで苦しんでいる

#### phase_4: レオンとリゼットの第一発見——指輪の回収

1. **教会の離れからの発見**
   - レオンとリゼットは教会の離れに滞在していた
   - 神父の絶叫を聞いて駆けつける
   - 二人が最初の発見者となる

2. **リゼットの冷静な行動**
   - レオンが回復魔法を使おうとするのを制止
   - 「私の方が得意だから」と言ってレオンを外に出す
   - レオンに人を呼びに行かせる

3. **リゼットと神父——二人きり**
   - リゼットは回復魔法を使わない
   - 冷たい声で神父に語りかける
   - **神父の指から「永劫の帯留」の指輪を素早く抜き取る**
   - ポケットに隠す
   - これにより、アンデッド化の物的証拠が消失

4. **リゼットの演技**
   - 廊下に向かって「誰か！ 神父様が……！」と叫ぶ
   - マリアが駆けつけるのを待つ
   - マリアが回復魔法を使うのを見届ける（計画通り）
   - マリアに疑いが向くよう演技する

#### phase_5: マリアの回復魔法——トドメの一撃

1. **マリアの善意**
   - リゼットの叫びを聞いて駆けつける
   - 苦しむ神父を助けようと回復魔法を使用
   
2. **回復魔法の逆効果**
   - 回復魔法は**聖属性**
   - アンデッド化した神父に対して猛毒となる（指輪の効果が残存）
   - 回復の光が触れた瞬間、神父の体が反応して発火
   - 急速に炭化して死亡
   
3. **ログに残らない証拠**
   - 教会の回復魔法は宗教上の規定でログに記録されない
   - マリアの証言を裏付ける記録がない
   - マリアへの疑惑が集中
   - リゼットは「マリア……あなた、何を……」と演技

### トリックの巧妙さ

1. **時間差攻撃**
   - リゼットは祝賀会に戻っており、完璧なアリバイ
   - 実際の殺害時刻と死亡時刻にズレがある
   
2. **マリアの善意を利用**
   - マリアの「暖房起動」が引き金
   - マリアの「回復魔法」がトドメ
   - 全てマリアのログに記録され、疑惑が集中
   
3. **ログの盲点**
   - 火の魔法のログは残る→マリアが疑われる
   - 回復魔法のログは残らない→マリアの証言が信じられない
   - 電気魔法のログは残る→しかし祝賀会中のリゼットには完璧なアlibi
   
4. **物理的証拠の偽装**
   - 電気熱傷を火傷に見せかける
   - 「癒しの風」による聖属性ダメージは検出困難
   - 指輪の存在に気づかなければ真相に辿り着けない

### ギルバートによる解明のポイント

1. **組織サンプルの分析**
   - 王立科学研究所の鑑識報告で判明
   - 火炎熱傷と電気熱傷の違い
   - 古い傷（火炎）と新しい傷（電撃）の区別
   
2. **暖房システムの調査**
   - 神父の部屋だけ寒かった理由の解明
   - 配管の弁が閉められていた証拠
   - 「癒しの風」の聖属性機能の発見
   
3. **指輪の発見**
   - 炭化した遺体から「永劫の帯留」の指輪を発見
   - リゼットが展示していたアイテムの一部
   - アンデッド属性の効果を解明
   
4. **リゼット自身の体温**
   - リゼットはアンデッドであり体温が低い
   - Scene 02でレオンが手を触って「冷たい」と感じる（削除済み）
   - アンデッドだからこそ、神父の部屋が寒いことに気づかなかった

### 解決パートでの展開

ギルバートは以下の順序で真相を明かす：

1. 組織分析の結果報告（電気熱傷の証明）
2. 暖房システムの異常（弁が閉められていた）
3. 「癒しの風」の聖属性効果
4. 指輪の発見とアンデッド化のメカニズム
5. リゼットのアリバイ崩しと犯行の再構成
6. マリアの完全な無実の証明
