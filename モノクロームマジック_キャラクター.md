# 主要キャラクター

## エルナ・ヴェルミリオン（Elena Vermillion）

### 基本情報
- **年齢**: 18歳（中高生向けターゲットに合わせて）
- **職業**: 魔導士（英雄パーティ所属）
- **身長**: 158cm
- **体重**: 45kg（栄養失調気味で細身）
- **髪色**: 漆黒のロングヘア（姉の白髪への対抗心から黒で固めている）
- **瞳の色**: 深紅（ストレスで充血気味、周囲には「深淵を覗きすぎた魔導士の瞳」と誤解される）
- **外見の特徴**: 
  - 顔色が常に悪い（栄養失調とストレス）
  - 全身を黒で統一したローブ（白の姉への対抗心）
  - つば広帽子（姉からもらったもの、呪いの象徴）
  - 細身で華奢な体型

### 性格
- **表の性格**: クールで孤高、無口で感情を表に出さない（見栄の仮面）
- **裏の性格**: 実は感情豊かで、見栄を張りたがる中二病気質
- **特徴的な行動**:
  - 借金取りを避けるためにキョロキョロ → 周囲には「隙のない警戒」と誤解
  - 報酬額を見てゴクリ → 「武者震い」と誤解
  - 黒マントで逃走 → 「かっこいい」と崇拝
  - 防御魔法をケチる → 「魔力温存の戦略」と誤解
- **口癖**: 「興味ないわ」（本当は興味あるけど、借金取りが来るから断る）
- **独白**: 中二病な必殺技名を小声で練習（「黒炎咆哮・ローンブレイズ」など）

### 能力・戦闘力
- **実際の魔法力**: 中程度（天才ではないが、実力はある）
- **見かけの魔法力**: 周囲には「深淵の闇を操る孤高の天才」と誤解されている
- **魔法の特徴**:
  - 手動詠唱（レガシー・マニュアル）を使用
  - 実は金がなくて最新デバイス（オート・スペル）が買えないだけ
  - 周囲は「便利さに頼らない本物の求道者」と誤解
  - 詠唱破棄に見えるが、実は高価な魔法スクロール（使い捨て）を砕いているだけ
- **魔法のエフェクト**: 
  - 灰色の煙と黒い煤（ジャンク魔石の不完全燃焼）
  - 周囲は「深淵の闇（モノクローム・ダークネス）」と誤解
- **装備**:
  - 杖：3億G（借金の原因、実際は使いこなせていない）
  - ローブ：5000万G（借金の原因）
  - 魔石：ジャンク魔石（300G）を常用、高純度魔石は質に入れている

### 借金の詳細
- **総額**: 3億G
- **返済期間**: 35年ローン
- **借金の原因**:
  1. 最新の魔石装備（3億Gの杖・高純度魔石）を購入
     - しかし実際は「まともな石」を買っても使いこなせず、クズ石で誤魔化している
  2. 新興魔法研究組織（スタートアップ）への投資
  3. 魔法商材（攻略本詐欺）への投資
     - 水晶玉配信者に騙されかけたり、逆に「成功者」として祭り上げられそうになる
  4. 水晶玉配信者の影響で「最新技術が必要」と信じ込まされた
  5. 王都の一等地の家賃（激高）

### 過去・経歴
- **幼少期**: 伝説の魔女セレス（姉）に憧れる
- **呪いの始まり**: 
  - セレスが魔王軍を殲滅した日、無表情でつば広帽子を幼いエルナに被せる
  - エルナの誤解：「あの日、姉さんは私に『後を託す』と言った（気がした）。だから私は天才でなければならない！」
  - 実は姉は極度のコミュ障で無表情だっただけ
- **現在**: 英雄パーティに所属し、借金返済に追われる日々

### 内面の葛藤
- **視界のモノクローム**: 
  - 借金のストレスと姉への劣等感で、視界は常に『色褪せたモノクローム』に見えている
  - 勇者に救われた瞬間に初めて『世界の色』を取り戻す（カラーになる）
- **仮面の二重構造**:
  - 見栄の仮面：借金を隠し、「天才」の仮面を維持
  - 子どもたちからの贈り物の仮面：守られるための仮面
- **心理的ジャーニー**:
  1. 呪いの継承：セレスの「無言」を「後を託す」と誤解
  2. 鎧の形成：借金を隠し、「天才」の仮面を維持
  3. 鎧の重さ：一人で背負い込み、友情を拒絶
  4. 鎧の破綻：暴走による全ての暴露
  5. 鎧の解除：弱さを認め、他者に分けてもらう

### 関係性
- **セレス（姉）**: 憧れと劣等感の対象、呪いの根源
- **勇者**: パーティリーダー、最終的にエルナを救済
- **パーティメンバー**: 距離を置いている（借金を隠すため）
- **子どもたち**: 路地裏で助けた子どもたちから仮面をもらう、唯一の温かさ

- **職業**: 勇者（英雄パーティリーダー）
- **身長**: 180cm
- **体重**: 75kg
- **髪色**: 金髪
- **瞳の色**: 碧色
- **外見**: 典型的な勇者風の外見、爽やかで頼りになる

- **基本性格**: 正義感が強く、仲間思い
- **価値観**: 「仲間は助け合うもの」「一人で背負い込む必要はない」
- **エルナへの態度**: 
  - エルナの「孤高の天才」という誤解を信じている
  - しかし、エルナが一人で背負い込んでいることに気づき始める
  - 最終的にエルナの借金を知り、肩代わりを申し出る

### 能力・役割
- **戦闘力**: 高い（勇者としての実力は本物）
- **役割**: パーティリーダー、前衛
- **エルナとの関係**: 
  - パーティに加入してから約1年
  - エルナの実力を高く評価している
  - エルナが一人で背負い込んでいることに違和感を感じている

### エルナ救済の動機
- **理由1**: エルナの火力は世界を救うために必要
- **理由2**: 仲間として、一人で背負い込ませたくない
- **理由3**: 「水くさい」という価値観（仲間同士で助け合うのが当たり前）

### セリフ例
- 「エルナ！ 広範囲結界を頼む！ 魔力は残ってるだろ！」
- 「あえてギリギリで見切って魔力を温存しているのか！？」
- 「くっ、また研究か。俺たちへの依存を避けて個を磨く……ストイックすぎる」
- 「聞いたぞ、エルナ。お前、3億も借金があったんだってな」
- 「なんで言わなかった！ 水くさい！」
- 「安心しろ。お前の火力は世界を救う。借金は俺たちが肩代わりしてやる！」
- 「その代わり、完済するまで魔王軍全滅させる勢いで働いてもらうからな！ 休む暇なんてねえぞ！」
- 「無理はするなよ。守る役目は分け合えばいい」

## キャラクター設定の補足

### エルナの視点
- **モノクロームの視界**: 借金のストレスで視界が色褪せて見える
- **カラーの視界**: 勇者に救われた瞬間、視界に色が戻る
- **仮面の意味**: 
  - 見栄の仮面：借金を隠すための仮面
  - 子どもたちからの贈り物の仮面：守られるための仮面

### 誤解の構造
- **エルナの行動** → **周囲の誤解**
  - キョロキョロ → 隙のない警戒
  - ゴクリ → 武者震い
  - 黒マントで逃走 → かっこいい
  - 防御魔法をケチる → 魔力温存の戦略
  - 手動詠唱 → 便利さに頼らない本物の求道者
  - 灰色の煙・黒い煤 → 深淵の闇を操る孤高の天才

### キャラクターの成長
- **エルナ**: 見栄の仮面 → 弱さを認める → 受け入れられる
- **勇者**: エルナの誤解を信じる → 真実を知る → 救済を申し出る
- **パーティ**: エルナとの距離 → 真実を知る → 受け入れる
